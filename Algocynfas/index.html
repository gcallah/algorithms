Algocynfas

This is a project to create self-graphing algorithms in Javascript using the HTML5 "canvas" element.

The name is in honor of the tiny Welsh coal-mining town of Algocynfas, where Prof. Callahan and his friend Anas met, while working 18-hour shifts extracting black diamond from the rocky Welsh hills.
